# Tantie Béné

Modern African street-food experience powered by AI.

## 🚀 Concept

Tantie Béné is a next-generation food brand combining:
- authentic African cuisine
- premium branding
- AI-powered menu recommendations

## 🍽️ Features

- Interactive digital menu
- Allergy & ingredient filtering
- AI food recommendations
- Smart pairing system
- Premium visual branding

## 🧠 AI Engine

The AI system helps users:
- choose meals based on preferences
- detect allergens
- suggest full meal combinations
- enhance customer experience

## 🛠 Tech Stack

Frontend: React
Backend: Node.js / Express
AI Layer: Prompt-based recommendation engine
Database: PostgreSQL / SQLite

## 📦 Structure

See folders:
- frontend/
- backend/
- ai-menu-engine/
- branding/

## 🎯 Goal

Turn traditional African street food into a premium digital dining experience.
